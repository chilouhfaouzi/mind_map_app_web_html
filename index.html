Format Instructions

To generate your mind map, please follow the Markdown format below:

# Mind Map Root
- Branch 1
  - Leaf 1.1
  - Leaf 1.2
- Branch 2
  - Leaf 2.1
  - Leaf 2.2

Each branch is preceded by a dash (-) and sub-branches are indented with two spaces.
